# Python OOP TEST

Conjunto de mini proyectos en Python que trabajan **OOP** y **JSON**

* **Lenguaje:** Python 3.14
* **Persistencia:** JSON
* **Conceptos:** OOP, CRUD, Type Hints, Modularidad.

### 1. ATM System
Simulador de cajero automático.
* **Features:** Validación de fondos, historial de transacciones, persistencia de saldo.
* **Archivos:** `atmclass.py`, `menuatm.py`

### 2. RPG Inventory
Sistema de gestión de inventario para juegos.
* **Features:** Cálculo algorítmico de valor total, categorización de ítems.
* **Archivos:** `inventoryclass.py`, `maingame.py`

### 3. To-Do List
Gestor de tareas clásico.
* **Features:** Marcado de estado completado/pendiente, edición de tareas.
* **Archivos:** `todoclass.py`, `menutodo.py`
